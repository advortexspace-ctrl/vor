# Advortex - Космический Минималистичный Визуал

Полностью адаптивный веб-сайт с космической эстетикой для демонстрации минималистичного рекламного визуала.

## Особенности

- **Полностью адаптивный дизайн** - работает на всех устройствах от мобильных до десктопов
- **Интерактивная космическая сцена** с анимированным гироскопом/вихрем
- **Динамическое звездное поле** с созвездиями
- **Панель управления** для настройки визуальных параметров
- **Современный UI/UX** с плавными анимациями и переходами
- **Оптимизированная производительность**
- **Кроссбраузерная совместимость**

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, CSS Variables, Animations)
- Vanilla JavaScript (без зависимостей)
- Google Fonts (Inter, Space Grotesk)
- Font Awesome иконки
